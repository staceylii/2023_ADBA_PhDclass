This is the Spring 2023 course repo for "Applied Bayesian Data Analysis for Research", taught by Nikhil Garg at Cornell.
Course website: https://cs6384-orie6217.github.io/Spring2023/. 

The beginning lectures are heavily inspired by the following course by Aki Vehtari: Bayesian Data Analysis, https://avehtari.github.io/BDA_course_Aalto/. The demos and "utilities_and_data" folder are from the following repository, copied over and lightly modified for this class's purpose: https://github.com/avehtari/BDA_py_demos.

These lectures use PyStan3. It can be installed via pip: https://pypi.org/project/pystan/.
